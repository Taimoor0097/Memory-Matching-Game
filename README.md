## 🧠 Memory Matching Game
A simple and interactive Memory Matching Game built using HTML, CSS, and jQuery. The goal of the game is to flip cards and find matching pairs while using memory and concentration skills.
This project demonstrates DOM manipulation, event handling, and basic game logic using jQuery.

# 🎮 Game Overview
The Memory Matching Game is a classic card game where players flip two cards at a time to find matching pairs. If the cards match, they remain open; otherwise, they flip back after a short delay.
The game continues until all pairs are matched.

# ✨ Features

Interactive card flipping animation
Randomized card positions each game
Match detection logic
Smooth UI using CSS styling
Built with lightweight jQuery for dynamic behavior
Simple and responsive layout

# 🛠️ Technologies Used
HTML5 – Structure of the game
CSS3 – Styling and animations
jQuery – Game logic and interactivity
